## ⚙️ Customization Reference

### Change Primary Color
```
Find: #06b6d4
Replace with: [your color]
```

### Change Button Style
```css
.stButton > button {
  padding: 0.875rem 2rem; /* Adjust padding */
}
```

### Disable Animations
```css
* {
  animation-duration: 0s !important;
  transition-duration: 0s !important;
}
```

### Adjust Blur Effect
```css
backdrop-filter: blur(20px); /* Change to: blur(10px) */
```
